4. UI GRADIENTS (CSS / FLUTTER)
-----------------------------------------------------------------

[A] Main Text Gradient (Hero Titles)
Direction: Right to Left (RTL) / Left to Right (LTR)
Colors: Shibl Crimson (#AF0C15) -> Rose 600 (#E11D48)
CSS: linear-gradient(to right, #AF0C15, #E11D48)

[B] CTA Button Gradient (Primary Action)
Direction: Right to Left
Colors: Shibl Crimson (#AF0C15) -> Shibl Crimson Dark (#8B0A11)
CSS: linear-gradient(to right, #AF0C15, #8B0A11)

[C] Card Background Gradients (Stages)
1. Style A (Light Red Mix):
   Colors: Red 50 (#FEF2F2) -> Rose 50 (#FFF1F2)
   CSS: linear-gradient(to bottom right, #FEF2F2, #FFF1F2)

2. Style B (Deep Rose Mix):
   Colors: Red 50 (#FEF2F2) -> Rose 100 (#FFE4E6)
   CSS: linear-gradient(to bottom right, #FEF2F2, #FFE4E6)
